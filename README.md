# Max Westmoreland Portfolio

Portfolio website built with React, Vite, and Tailwind CSS.

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Framer Motion

## Project Setup

Open the project folder in VS Code.

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

After that, Vite will print a local URL in the terminal, usually:

```bash
http://localhost:5173/
```

Open that URL in your browser to view the site.


##Shut Down the Site Safely##

To stop the local development server:
click into the terminal where npm run dev is running
press Control + C

If the terminal asks for confirmation, type:

```bash
y
```

and press Enter.

This only stops the local server. It does not delete or damage any project files.


##Build for Production##

Create an optimized production build:

```bash
npm run build
```

This generates a dist/ folder with the production-ready site files.


##Preview the Production Build Locally##

To preview the built version locally:

```bash
npm run preview
```

Vite will print a local preview URL in the terminal.


To stop the preview server, press:

```bash
Control + C
```
